Descripción del Proyecto

Sitio web de fotografía con diseño minimalista y galería interactiva optimizada para carga rápida de imágenes. Incluye sistema de gestión de galerías, optimización automática de imágenes y experiencia de usuario fluida.

El proyecto implementa técnicas avanzadas de lazy loading, compresión de imágenes sin pérdida de calidad y navegación intuitiva para destacar el trabajo fotográfico de manera profesional.
